ManyChat Integration

Configure your ManyChat webhook to connect with our system.

Use this URL in your ManyChat webhook settings.
Add this token to your webhook headers for authentication.
Click to generate a new unique webhook token if needed.

Setup Steps:

  1. Go to your ManyChat dashboard
  2. Navigate to Settings > Integrations > Webhooks
  3. Add a new webhook with the URL above
  4. Set the content type to "application/json"
  5. Add a header "Authorization" with the token above
  6. Save your settings

n8n Integration

Configure n8n to process your webhook data.

Enter the webhook URL provided by your n8n instance.
Add this token to your n8n webhook node for authentication.

Request Format (ManyChat to System):

{
  "user": {
    "id": "user_12345",
    "name": "John Doe"
  },
  "message": {
    "text": "Hello, I need help with...",
    "timestamp": "2023-07-15T14:30:00Z"
  },
  "metadata": {
    "platform": "facebook",
    "source": "messenger"
  }
}

Response Format (System to n8n):

{
  "success": true,
  "conversation_id": "conv_67890",
  "response": {
    "text": "Thank you for your message. Our system is processing your request.",
    "timestamp": "2023-07-15T14:30:05Z"
  },
  "action": "process_request",
  "status": "pending"
}
Use these JSON structures as reference for your webhook integration.

Setup Steps:

  1. Open your n8n instance
  2. Create a new workflow or edit an existing one
  3. Add a "Webhook" trigger node
  4. Copy the webhook URL from n8n and paste it above
  5. Add the authentication token to your workflow
  6. Save your workflow and activate it
